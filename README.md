<h1 align="center">VDXExternal</h1>

<p align="center">
  Research project exploring external game tooling techniques for Roblox Violence District.
</p>

---

## Features

- External architecture
- Overlay rendering
- ESP framework
- Camera utilities
- Configuration system
- High-performance memory access
- Modular design
